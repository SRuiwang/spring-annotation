此demo是基于spring实现业务分离
    实现思想：
        在spring初始化所有类之前，将业务模块类扫描存入一个执行管理器中
        使用时只需要根据模块的方法以及子模块（命令）查找该方法进行调用即可
    技术：
        使用spring生命周期类 BeanPostProcessor
        该类在bean初始化前会被调用，我们扫描自定义注解的业务模块类
        保存在InvokerHolder
        调用时 根据 模块and命令查找，直接调用即可，实现了业务分离
